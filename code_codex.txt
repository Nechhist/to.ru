Правила для наименований объектов для сайта Турнир-онлайн

I. PHP

1. Переменные
- с нижним подчеркиванием
- первое слово - значение переменной
- второе слово - для чего
- второе слово - где находиться
- пример, $array_random_tour || $name_admin_forView

2. Функции
- верблюдом
- первое слово - что делаем
- второе слово - с чем делаем
- третье слово - для чего делаем
- пример, updateCollorForTb(), deleteDateOldTour()


II. JS
-все верблюдом
-остальное как в PHP


III. CSS
1. Название классов
-через тире -
-первое слово - что за html объект
-второе слово - какой логическую сущность обособляет
-третье слово - для чего создается или чем выделяется или где находится
-пример, div-news-main, span-time-for-news
